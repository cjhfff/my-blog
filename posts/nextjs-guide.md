# Next.js 入门指南

这是一篇关于 Next.js 的入门教程。

## 什么是 Next.js？

Next.js 是一个 React 框架，提供了：

- 服务端渲染 (SSR)
- 静态站点生成 (SSG)
- API 路由
- 文件系统路由

## 为什么选择 Next.js？

1. 性能优化
2. SEO 友好
3. 开发体验好
4. 生态系统完善
